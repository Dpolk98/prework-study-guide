Git

  • git status: checks what branch you are currently working in
  • git checkout -b "branch-name": creates a new branch with name: "branch-name", and switches to it
  • git push "branch-name": pushes the local commits to the remote branch
  • git pull: fetches and merges any commits from the remote branch
  • git clone "url": fethces and recreates a repo from the specified url
  • git pull origin main: same as git pull, but specifically for the main branch
