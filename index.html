CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • the . or * prefix will apply the command to every instance of the class it describes
  • CSS documents can be stored inline, internally, or externally
  • CSS can be stored anywhere, as long as you can link to it, you can apply it to your HTML file

Git

  • git status: checks what branch you are currently working in
  • git checkout -b "branch-name": creates a new branch with name: "branch-name", and switches to it
  • git push "branch-name": pushes the local commits to the remote branch
  • git pull: fetches and merges any commits from the remote branch
  • git clone "url": fethces and recreates a repo from the specified url
  • git pull origin main: same as git pull, but specifically for the main branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions must be called or they will not function
  • Arrays can hold any type of data (string, boolean, etc)
  • For loops must contain 3 statements: an initilization, a condition, and an iteration
